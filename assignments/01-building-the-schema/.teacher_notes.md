## What the Learner Needs to Do

Create one or two migration files that produce the following schema:

**products**
- `name` — string, NOT NULL
- `description` — text, nullable
- `price` — decimal, NOT NULL
- `stock_count` — integer, NOT NULL, default: 0
- An index on `price`
- Timestamps

**categories**
- `name` — string, NOT NULL
- A unique index on `name`
- Timestamps

The learner does not need to touch any model file for this assignment.

## Common mistakes

- Forgetting `null: false` on `name`, `price`, or `stock_count`
- Forgetting `default: 0` on `stock_count`
- Forgetting the unique index on `categories.name`
- Using `float` instead of `decimal` for `price` (specs will catch this)
- Running specs before running `rails db:migrate`
